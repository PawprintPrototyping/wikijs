# Pawprint Web Services


- https://wiki.pawprint.space - You're here! General documentation for all things Pawprint.
- https://members.pawprint.space - Pawprint's Member Matters instance. This is where members can manage their membership, report issues, and find helpful links.
- https://hass.pawprint.space - Home Assistant. This can be used to unlock the door if you forget your badge, control certain lights at the space, and to see some stats from sensors within the space.
- https://ipa.pawprint.space - FreeIPA. Allows members to change their LDAP user info and password (logins for the office computer, laser computer, etc.) - Accessible only from the Pawprint LAN.
- https://moodle.pawprint.space - Moodle Learning Management System. Hosts various self-guided learning content, both for pawprint-specific subjects and general topics.
- https://events.pawprint.space - Mobilizon events platform. Allows for managing events at the space.
- https://flightfeeder.pawprint.space - FlightFeeder, watch the airplanes over the space.
- https://netbox.pawprint.space - NetBox, knowledge base of Pawprint IP space.
- https://grafana.pawprint.space - Misc metrics.
